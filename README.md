# FikolasAI website

Site vitrine bilingue de FikolasAI, publié via GitHub Pages sur https://fikolasai.com/.

## Objectif

Présenter les offres FikolasAI autour de trois axes :

- FikolasAI Academy : formation IA personnalisée et mentorat actionnable.
- FikolasAI Agent : co-création d'agents IA pour automatiser des process métier.
- FikolasAI Studio : système d'agents IA interconnectés pour solopreneurs et PME.

Le site met aussi en avant Cyril Cieslak, son approche pédagogique, des exemples d'agents IA, les profils clients visés, le process d'accompagnement, une FAQ, des partenaires issus de la première version du site et les appels à l'action vers le formulaire Tally.

## Structure

- `index.html` : page principale du site, avec contenu français/anglais, styles Tailwind via CDN et script de changement de langue.
- `assets/` : images utilisées sur le site.
- `CNAME` : configuration du domaine personnalisé `fikolasai.com`.

## Contact

Les boutons principaux pointent vers le formulaire Tally : https://tally.so/r/2EbyZL.

## Publication

Le site est déployé automatiquement par GitHub Pages lorsque la branche `main` est mise à jour.
